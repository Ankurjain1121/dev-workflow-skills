# Frontend Design

Create distinctive, production-grade frontend interfaces.

## Features
- **Anti-Slop Rules** - Avoids generic AI aesthetics
- **High Design Quality** - Polished, professional code
- **shadcn/ui Integration** - Component generation support
- **Production Ready** - Code you can ship

## Usage

```bash
/frontend-design "Build a landing page for a SaaS product"
/frontend-design "Create a pricing table component"
```

## Installation

Enable in your Claude settings:
```json
{
  "enabledPlugins": {
    "dev-workflow-skills@frontend-design": true
  }
}
```
